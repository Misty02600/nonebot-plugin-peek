# Progress: nonebot-plugin-peek

## 已完成功能 ✅

### v0.2.0 多主机支持 (2026-02-10)

- [x] 多主机配置
  - `PEEK_HOST` 支持逗号分隔多主机
  - `peek_hosts` 属性自动解析为列表
  - 向后兼容单主机配置

- [x] 智能主机选择
  - `HostManager` 类管理多主机
  - 并发查询各主机 `/idle` 端点
  - 自动选择空闲时间最短（最近操作）的主机

- [x] 服务层扩展
  - `IdleInfo` 数据类
  - `get_idle_info()` 方法
  - `host` 属性（用于日志显示）

### v0.1.0 重构 (2026-02-08)

- [x] 项目结构重组
  - `const.py` - 常量定义
  - `config.py` - 配置模型
  - `service.py` - PeekAPI 客户端
  - `dependencies.py` - 依赖注入
  - `handler.py` - 命令处理器
  - `__init__.py` - 插件入口

- [x] 配置系统
  - 单下划线前缀 (`PEEK_*`)
  - Pydantic 模型验证
  - 完整配置文档

- [x] 命令系统
  - `/peek` - 获取模糊截图
  - `/peek 原图` - 超级用户获取原图
  - `/peep` - 获取音频录制
  - `/peek_check` - 检查服务状态

- [x] 依赖管理
  - `httpx` - HTTP 客户端
  - `nonebot-plugin-localstore` - 数据存储

- [x] 代码质量
  - Ruff 检查通过
  - Pyright 类型检查通过
  - 完善的文档注释
