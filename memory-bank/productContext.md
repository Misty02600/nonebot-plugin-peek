# Product Context: nonebot-plugin-peek

## 项目存在的原因

这个插件为 NoneBot2 提供了一个有趣且实用的功能：让群友可以"视奸"主人的电脑。这可以用于：
- 向群友展示当前正在做什么
- 远程监控电脑状态
- 增加群聊互动的趣味性

## 解决的问题

1. **远程查看需求**: 用户可能需要在不在电脑前时，通过 QQ 查看电脑屏幕
2. **隐私保护**: 通过模糊半径功能，可以在分享屏幕的同时保护敏感信息
3. **权限控制**: 超级用户可以获取原图，普通用户只能获取模糊图
4. **状态通知**: 主人可以收到谁在查看电脑的通知

## 工作原理

### 架构流程

```
用户发送命令 → NoneBot2 → 插件处理 → 请求本地服务器 → 返回截图/音频 → 发送给用户
```

### 状态码处理

| 状态码   | 含义       | 响应                                  |
| -------- | ---------- | ------------------------------------- |
| 200      | 成功       | 返回截图/音频                         |
| 401      | 未授权     | 返回 "Hダメだ!死刑!"                  |
| 403      | 禁止访问   | 返回 "主人正在瑟瑟，过段时间再试试吧" |
| 500/错误 | 服务器错误 | 返回 "电脑未开启或下载图片失败"       |

### 用户权限

- **普通用户**: 只能获取模糊后的截图
- **超级用户**: 可以使用密钥获取原图，或使用 `peek 原图` 命令

## 用户体验目标

1. **简单易用**: 一个命令即可获取截图
2. **响应迅速**: 优化下载和超时机制
3. **友好反馈**: 各种状态都有对应的提示和备用图片
4. **隐私可控**: 模糊功能保护敏感内容
